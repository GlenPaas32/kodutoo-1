# Digitaalne Kell

## Autor

## Kirjeldus
Veebirakendus, mis kuvab:
- kellaaega (tunnid, minutid, sekundid)
- kuupäeva (päev, kuu, aasta)
- nädalapäeva

Funktsionaalsus:
- Fondi suuruse muutmine (+ ja - nupud ning klaviatuur)
- Päeva/öö režiim automaatselt
- EventListenerid ja funktsioonid kasutusel

## Lisafunktsioonid
- Teema muutub vastavalt ajale
- saab ka ise muuta öö/päeva vahel
